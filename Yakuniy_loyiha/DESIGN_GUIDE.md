## 🔄 Dark/Light Mode Support

All colors automatically adapt:

```cpp
// Access current mode
if (AppSettings::DarkMode) {
    // Use dark-optimized colors
} else {
    // Use light-optimized colors
}

// All color getters handle this internally
Color primary = AppSettings::PrimaryColor();  // Auto-adjusts
Color bg = AppSettings::PageBackColor();      // Auto-adjusts
```

**Dark Mode Benefits:**
- Easier on eyes in low-light conditions
- Vibrant accent colors pop more
- Professional appearance

**Light Mode Benefits:**
- Clean, fresh appearance
- Great for presentations
- Accessible in bright environments

## 🎯 Implementation Guide

### Using Premium AppSettings
```cpp
// Apply full theme
AppSettings::ApplyTheme(form);

// Premium button styling
AppSettings::StyleModernButton(btn, true);   // Primary
AppSettings::StyleModernButton(btn, false);  // Secondary

// Create premium cards
Panel^ card = AppSettings::CreatePremiumCard(width, height, radius, bgColor);

// Style components
AppSettings::StyleInput(textBox);
AppSettings::StyleGrid(dataGridView);

// Access gradients
Color gradStart = AppSettings::GradientPrimaryStart();
Color gradEnd = AppSettings::GradientPrimaryEnd();
```

### Creating Premium Components

**Premium Header:**
```cpp
Panel^ header = gcnew Panel();
header->Size = System::Drawing::Size(width, 220);
header->BackColor = AppSettings::PrimaryColor();
// Add gradient background (see AboutForm)
```

**Premium Card:**
```cpp
Panel^ card = gcnew Panel();
card->BackColor = AppSettings::SurfaceColor();
AppSettings::MakeRounded(card, 16);

// Add accent bar
Panel^ accentBar = gcnew Panel();
accentBar->BackColor = AppSettings::GradientPrimaryStart();
card->Controls->Add(accentBar);
```
